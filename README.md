# Customer Churn Prediction — BCG Data Science Simulation (Forage)

This project was completed as part of the BCG Data Science Virtual Experience Program on Forage. The objective was to analyze customer data from PowerCo (energy provider for SMEs) and build a machine learning model to predict customer churn.

Customer churn prediction helps companies identify at-risk customers and take proactive retention actions to reduce revenue loss.

## Approach
- Performed exploratory data analysis (EDA) to understand customer behaviour
- Conducted data preprocessing and feature engineering
- Built a Random Forest classification model
- Evaluated model using accuracy, precision, recall, F1-score, and ROC-AUC

## Result
- ROC-AUC: 0.65
- Recall: 0.27
- Accuracy: 0.83

Results indicate moderate predictive ability, while highlighting the challenge of predicting churn due to class imbalance (9.7% churn rate).
